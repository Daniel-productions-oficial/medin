# medin:
medin es un modelo de detecsion de intencion como parte de los modelos de ai entrenados por daniel de anda para el desarollo de nova, el asistente de ai creado por daniel de anda un desaroyador jr.
<br>
**Medin** es un modelo de detección de intención, lo que significa que identifica la intención del usuario a partir del texto que se le proporciona. Medin es un modelo de IA diseñado específicamente para un dispositivo llamado Nova, un asistente similar a Alexa.
## que es nova y ¿como nova nesecita de medin?
Nova es un dispositivo tipo asistente que utiliza un modelo de IA llamado Kamutini, basado en un modelo de Qwen. Kamutini es un modelo cuantizado de 5 mil millones de parámetros. Nova usa Medin para detectar la intención del usuario, ya que sin 
este sistema sería imposible determinar si el usuario quiere investigar o simplemente charlar.
